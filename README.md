# 🚀 Birendra Joshi - Portfolio

A modern personal portfolio website built using **Next.js, TypeScript, and Tailwind CSS**.

This project showcases my skills, projects, experience, and contact information in a clean, responsive, and animated UI.

- Photo gallery
- Contact form
- Modern UI/UX design

## 📁 Project Structure

- `/app` - Pages and routing (Next.js App Router)
- `/components` - Reusable UI components
- `/public` - Images and static assets
- `/styles` - Global styles
